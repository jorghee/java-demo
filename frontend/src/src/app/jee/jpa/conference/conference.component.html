From JPA to JSON how to show the data from your database

The events calendar is loaded using JPA from an H2 (file) database.
In the first version of this demo the data is stored in sql files and inserted in the database by the container during the deployment phase.
This solution is particularly beneficial to create sample database for the integration tests and for demos like this one.
